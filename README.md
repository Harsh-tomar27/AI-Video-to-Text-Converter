# 🎥 AI Video-to-Text Converter

An AI-powered web application that converts speech from video files into text using the **OpenAI Whisper Base Model**. The application extracts audio from uploaded videos, transcribes spoken content, and generates a downloadable text transcript through a simple Streamlit interface.

## 🚀 Features

- Upload video files (`.mp4`, `.avi`, `.mov`, `.mkv`)
- Automatic Speech-to-Text conversion
- Powered by OpenAI Whisper (Base)
- Display transcript in the browser
- Download transcript as a `.txt` file
- Simple and user-friendly Streamlit interface

---

## 🎯 Project Objective

The objective of this project is to automate the process of converting speech in videos into text. It eliminates manual transcription, saves time, and provides an accurate and efficient solution using a pre-trained AI model.
---
## ⚙️ Tech Stack

- **Python** – Backend development
- **Streamlit** – Web application
- **OpenAI Whisper (Base)** – Speech-to-Text model
- **PyTorch** – Deep Learning framework
- **MoviePy** – Audio extraction
- **FFmpeg** – Multimedia processing
